# Amirreza Haghani Portfolio

Modern animated personal portfolio built with Vue 3, TypeScript, GSAP, Pinia, Vue Router, and Tailwind CSS.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Stack

- Vue 3 Composition API
- TypeScript
- GSAP animations
- Pinia
- Vue Router
- Tailwind CSS
